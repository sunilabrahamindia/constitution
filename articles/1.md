## Constitutional Text

<div id="fulltext" class="fulltext" markdown="1">
**(1)** *India, that is Bharat, shall be a Union of States.*

**(2)** *The States and the territories thereof shall be as specified in the First Schedule.*

**(3)** *The territory of India shall comprise —*  
**(a)** *the territories of the States;*  
**(b)** *the Union territories specified in the First Schedule; and*  
**(c)** *such other territories as may be acquired.*
</div>

## Summary

- India has two constitutional names: **India** and **Bharat**.  
- The Constitution describes India as a **Union of States**, signalling permanence and non-secession.  
- The First Schedule lists States and Union Territories.  
- Clause (3) allows constitutional acquisition of new territory.

## Explanation

Article 1 sets the foundation of India's political geography. The phrase **Union of States** was chosen to indicate an indestructible Union rather than a federation formed by compact. Clause (3) provides an expandable territorial definition that can accommodate internal reorganisation and lawful acquisitions.

## Historical Context

Constituent Assembly debates show the framers' careful balancing of tradition and international continuity when settling on the dual name. The language prioritised unity while preserving flexibility for future reorganisation.

## Key Judicial Interpretation

### Re: Berubari Union (1960)
The Supreme Court held that cession of Indian territory to another sovereign requires a constitutional amendment.

### State Reorganisation Cases
Rulings have reiterated Parliament’s wide power under Articles 2–4 to organise, create, merge, or alter States without altering Article 1.

document.addEventListener('DOMContentLoaded', () => {
  const title = (document.querySelector('meta[property="og:title"]')?.content
                 || document.title
                 || document.querySelector('h1')?.innerText
                 || 'article').toString().trim().toLowerCase().replace(/\s+/g, '-').replace(/[^a-z0-9\-]/g, '');

  /* COPY full text */
  document.querySelectorAll('.copy-btn').forEach(btn => {
    btn.addEventListener('click', async () => {
      const target = document.querySelector(btn.dataset.copytarget || '#fulltext');
      if (!target) return;
      const original = btn.textContent;
      try {
        await navigator.clipboard.writeText(target.innerText.replace(/\u00A0/g,' ').trim());
        btn.textContent = 'Copied';
        setTimeout(()=> btn.textContent = original, 1200);
      } catch (e) {
        btn.textContent = 'Copy failed';
        setTimeout(()=> btn.textContent = original, 1200);
      }
    });
  });

  /* DOWNLOAD plain text (.txt) */
  const dlTxt = document.querySelector('.download-txt');
  if (dlTxt) {
    dlTxt.addEventListener('click', (ev) => {
      ev.preventDefault();
      const target = document.querySelector('#fulltext');
      if (!target) return;
      const text = target.innerText.replace(/\u00A0/g,' ').trim();
      const blob = new Blob([text], { type: 'text/plain;charset=utf-8' });
      const url = URL.createObjectURL(blob);
      const a = document.createElement('a');
      a.href = url;
      a.download = `${title || 'article'}.txt`;
      document.body.appendChild(a);
      a.click();
      a.remove();
      URL.revokeObjectURL(url);
    });
  }

  /* DOWNLOAD markdown (.md) - attempt to create clean markdown from innerText */
  const dlMd = document.querySelector('.download-md');
  if (dlMd) {
    dlMd.addEventListener('click', (ev) => {
      ev.preventDefault();
      const target = document.querySelector('#fulltext');
      if (!target) return;
      // We use innerText to avoid HTML tags; this yields readable Markdown-ready text.
      // For more advanced conversions we can integrate a proper HTML->MD converter later.
      const md = `## ${document.title || 'Article'}\n\n` + target.innerText.replace(/\u00A0/g,' ').trim();
      const blob = new Blob([md], { type: 'text/plain;charset=utf-8' });
      const url = URL.createObjectURL(blob);
      const a = document.createElement('a');
      a.href = url;
      a.download = `${title || 'article'}.md`;
      document.body.appendChild(a);
      a.click();
      a.remove();
      URL.revokeObjectURL(url);
    });
  }

  /* PRINT */
  const pr = document.querySelector('.print-article');
  if (pr) {
    pr.addEventListener('click', (ev) => {
      ev.preventDefault();
      window.print();
    });
  }
});
</script>
